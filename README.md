Delta Exchange API Client for Elixir

## Installation

Add the `ex_ftx` package to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:ex_delta_exchange, "~> 0.0.1"}
  ]
end
```

## Requirements

- Erlang 22+
- Elixir 1.10+
